## 📷 실제 화면 캡처 목록 (dflo)

> 본문엔 "이미 개발했다"는 표현을 쓰지 않고, **개발 목표 UI의 예시 화면**으로 배치합니다.

### [그림 2] 성장 예측 결과 화면  *(2장 2-1)*
- **어디서:** 예측키 계산기 결과 화면 (웹 `/calc.html` 또는 홈페이지 예측키 팝업)
- **무엇을:** 국가표준 성장 백분위 곡선(P3·P50·P97) + 현재 키 + 예측 성인키가 보이는 결과 화면
- 가급적 **숫자·곡선이 또렷하게** 나온 상태로 캡처

### [그림 4] 의료진용 진료 지원 화면 (2~4컷)  *(2장 2-2)*
어드민 환자 상세에서 아래 중 2~4개를 캡처해 배치:
1. **환자 성장곡선 그래프** — 백분위 곡선 + 뼈나이 회차 + 예측 추세 (AdminPatientGrowthChart / PredictedHeightTrend)
2. **X-ray 뼈나이 분석 뷰어** — 줌·마킹 들어간 화면 (ZoomableImg)
3. **다중검사 기반 처방 추천 화면** — 추천 결과 + 근거(논문) 표시 (RxRecommendModal)
4. **환자 진료 대시보드** — 진료내역/X-ray/Lab/생활습관 탭이 보이는 3-column 화면 (VisitDetailPanel)

### [그림 6] 글로벌 다국어 서비스 화면  *(5장 5-1)*
- **무엇을:** 한국어/태국어/베트남어/영어 홈페이지 화면(`/ko` `/th` `/vi` `/en`)을 나란히 + 치료사례 성장그래프(`/cases`)
- 글로벌 확장 가능성의 시각적 근거
